## Audit Methodology

All audits follow a read-only-first approach:
1. Static analysis of all source layers
2. Test baseline execution
3. Design doctrine cross-check
4. Findings register → remediation plan → execution

## Inventory

The [docs-inventory.md](docs-inventory.md) is the authoritative map of all documentation.
It classifies every .md file in docs/ by category, status, and recommended action.
