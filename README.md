# PFIOS

PFIOS（Personal Financial Intelligence Operating System）是一个围绕金融分析、治理、审计、报告、建议跟踪与复盘学习构建的个人金融智能操作系统。

## Core Architecture

Interface → API → Orchestrator → Reasoning → Governance → Expression → Object/Knowledge → Core/Persistence

## Principles

- 推理不等于治理
- 治理不等于审计
- 表达不等于持久化
- Skills 不等于 Wiki
- Runtime 不等于系统真相源

## Step 1 Goal

完成最小可运行骨架：
- FastAPI 启动
- Web 控制台启动
- Mock analyze 主链打通
- 核心边界固定
